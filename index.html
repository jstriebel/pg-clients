Usage

You can connect to multiple postgres servers:
  • postgres: a dockerized postgres instance for testing,
    the defaults are user:postgres, password:postgres, database:db
  • 172.17.0.1: your local postgres instance (usually localhost),
    only works if remote connections are allowed,
    or you have to add network_mode: host in docker-compose.yaml.
The default postgres port is 5432.
